注意：

  1. ./configure：测试配置环境

  3. make install 就是安装的意思
  4. make && makeinstall的意思就是执行make如果没有发生错误就执行make install

七、验证安装是否正确

  1. 验证python，如果显示的版本号是3.7.0，就代表安装成功了（CentOS会自带一个python2.7) python –version
  2. 验证pip，看看pip的版本号是不是10.0.1，以及最后括号里是不是3.7 pip –version
